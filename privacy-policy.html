Requestly Chrome Extension - Privacy Policy

Last updated:

Important: Requestly does not collect, store, or transmit any personal user data. This extension operates entirely locally within your browser.

1. Data Collection

Requestly does not collect any personal information from users. The extension operates without gathering:

  • Personal identification information (name, email, address)
  • Browsing history or web activity
  • Document content or file information
  • User authentication credentials
  • Location data or device information

2. Local Storage

The extension only stores minimal user preferences locally using Chrome's built-in storage API:

  • Extension enabled/disabled setting: Whether the auto-request feature is active
  • Custom message text: User-defined message for access requests (default: "Hi, could you grant me access to this file? Thanks!")

This data remains on your device and is not transmitted to any external servers.

3. How Requestly Works

Requestly operates by:

  • Monitoring Google Docs and Google Drive pages for access-restricted content
  • Automatically detecting "Request access" buttons when they appear
  • Filling in optional message fields with your custom text
  • Clicking request buttons to streamline the access request process

All operations occur locally within your browser without external data transmission.

4. Third-Party Services

Requestly does not integrate with any third-party services or external APIs. The extension only interacts with Google Docs and Google Drive websites as necessary for its core functionality.

5. Data Security

Since Requestly does not collect or transmit user data, there are no data security concerns. All user preferences are stored locally using Chrome's secure storage mechanisms.

6. Children's Privacy

Requestly does not collect any information from users of any age, including children under 13.

7. Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last updated" date.
